# Validation of Results

The results generated by the pipeline were validated through manual inspection.

## Abstract Extraction

Extracted abstracts were compared with the original PDF documents for several papers to verify correctness.

## Figure Detection

The number of figures was obtained by counting `<figure>` elements in the TEI XML generated by GROBID.

These counts were compared with the figures visible in the original papers.

## Link Extraction

Extracted URLs were manually inspected to ensure they correspond to valid external resources mentioned in the papers.

## Limitations

Some limitations arise from the automatic parsing of PDFs:

- figures may be over-detected depending on document structure
- URLs may be truncated if broken across lines
- some references may not be detected as links
- poorly formatted PDFs may reduce extraction accuracy
