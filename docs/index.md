# OSAI Paper Analysis Pipeline

This documentation describes the software developed for the **Open Science and AI course project**.

The project implements a reproducible pipeline for analysing scientific papers using **GROBID** and Python.

## Project Overview

The pipeline processes a set of open-access research papers and extracts structured information such as:

- Abstract text
- Number of figures per paper
- External links found in the documents

The extracted information is used to generate visualisations and structured datasets.
